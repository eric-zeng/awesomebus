AwesomeBus

An interactive visualization of Seattle-area public transportation.

Tutorial

  • Zoom in to view bus stops
  • Click on a route or type in a route number to see where it goes
  • Click on stops to see its routes and real time arrival data
  • Select all routes in an area using the shape tool
  • Double-click on a route to see transfers

See real-time arrival data at OneBusAway
